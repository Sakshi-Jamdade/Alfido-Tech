# Alfido-Tech

📂 Project Structure
Task 1/ - Banking System: Demonstrates OOP concepts like Inheritance and Encapsulation.

Task 2/ - Array Operations: Demonstrates loops and conditional logic.

Task 3/ - Collections & Exceptions: Implements ArrayList, HashMap, and Custom Exceptions.

Task 4/ - File Handling: Demonstrates reading and writing to .txt files using FileReader and FileWriter.

🛠️ Built With
Java - The programming language used.

Git/GitHub - Version control and hosting.

📝 Key Concepts Covered
Object-Oriented Programming (OOPs)

Java Collections Framework (List, Set, Map)

Exception Handling (Try-Catch-Finally)

File Input/Output (I/O)
